# 基本格式

```makefile
target ... : prerequisites ...
	command
```



| automatic variables |                      descriptions                      |
| :-----------------: | :----------------------------------------------------: |
|        `$@`         |                       目标文件名                       |
|        `$<`         |            第一个先决条件(文件)的名称(常用)            |
|        `$^`         |      所有先决条件的名称(去重, 每个规则只出现一次)      |
|        `$+`         | 规则中所有先决条件列表(包括出现一次的和重复出现多次的) |

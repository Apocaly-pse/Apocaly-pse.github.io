# 字符串创建

## Python

```python
s1='1'
s2='abc'
s3="45"
s4=r"D:\Desktop"
```

单双引号都可以, 并且支持原生(raw)字符串. 

>   小坑:
>
>   原生字符串不支持单字符, 即`s=r'\'`这样的写法是会报错的, 只能老老实实转义:`s='\\`.



## C++

```cpp
string s1="123";
// string s2='1';// error
string s3{"abc"}; //initializer_list, C++11
string s4=u8""; //UTF-8字面量
```

## `sed`





提取IP地址

```bash
❯ ifconfig en0 | sed -n '5p' | sed -e 's/^.*inet//g' -e 's/net.*$//g'
 192.168.0.103
```



```bash
❯ ifconfig en0 | sed -e '5s/^.*inet//' -e "5s/netmask.*$//p" -n
 192.168.0.103
```





## `awk`命令



提取IP地址

```bash
❯ ifconfig en0 | awk 'NR==5{print $2}'
192.168.0.103
```
